# To-Do_List_App
A Detailed To-Do List App with Basic Authentication

provides the following information for your task:
- Timestamp
- Title
- Description
- Due Date
- Tags
- Status

The following REST APIs were created using DRF (DjangoRestFramework):
- CREATE a to-do item
- READ one to-do item
- READ all to-do items
- UPDATE a to-do item
- DELETE a todo item
